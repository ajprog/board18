Welcome to the Board18 Project.

BOARD18 is a tool for playing “18xx style” games via Email.

The BOARD18 project's mission is to create a tool which will assist players of 18xx games by displaying in a web application game boards and stock charts on which track tiles and tokens can be easily manipulated by the players.

The BOARD18 game session associated with a game is used to display the game board and the stock market for the game. It is run as a web service accessed via the player's web browser.

BOARD18 is open source software available under the MIT License.

For more information on 18xx style games, see this wikipedia article.
